測試書籍說明
================

這個資料夾包含測試用的 EPUB 檔案。

## 如何獲取測試檔案：

1. **免費 EPUB 檔案來源：**
   - Project Gutenberg: https://www.gutenberg.org/
   - Internet Archive: https://archive.org/
   - 標準文件庫: https://standardebooks.org/

2. **建議的測試檔案：**
   - 愛麗絲夢遊仙境 (Alice's Adventures in Wonderland)
   - 簡愛 (Jane Eyre)
   - 任何中文 EPUB 檔案

3. **檔案放置位置：**
   - 將 EPUB 檔案放在此 test_files 資料夾中
   - 然後在模擬器中使用應用程式的匯入功能

## 測試功能清單：

- [x] 匯入 EPUB 檔案
- [x] 建立書架
- [x] 閱讀進度記錄
- [x] 主題切換
- [x] 書籤功能
- [x] 搜尋功能
